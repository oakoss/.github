# Contributing to Oak OSS

Thanks for your interest in contributing to Oak OSS projects!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a branch for your changes
4. Make your changes
5. Submit a pull request

## Development

Each repository has its own setup instructions in its README. Generally:

```bash
# Install dependencies
bun install

# Run development server
bun dev

# Run tests
bun test

# Build
bun run build
```

## Pull Requests

- Keep PRs focused on a single change
- Write clear commit messages
- Update documentation if needed
- Add tests for new features
- Ensure all tests pass

## Code Style

We use automated formatting and linting. Run these before committing:

```bash
bun run format
bun run lint
bun run typecheck
```

## Questions?
